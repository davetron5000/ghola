This site was created by hand. The source code is available on GitHub. Various libraries and technologies were used to assist.

MelangeCSS is used for all styling, using its default palette, which you can view in Ghola itself.

The JavaScript is mostly vanillaJS, but with a library I created called BrutalDOM that provides a way to create lightweight wrappers around DOM elements. This makes it easier to wire up events that trigger behavior.

chroma.js is used to manipulate and model the colors. The color names themselves come from hex-color-to-color-name".

The dev environment is powered by Docker, esbuild, and Chokidar CLI.

The font is Helvetica Neue, if you have it as part of your system. Otherwise, the site attempts to use Inter, then Robot, then Nimbus Sans, then falls back to the system sans-serif. The code font is Courier, if available (otherwise the system monospace is used).

The name “Ghola” is from the Dune series. A ghola is an artificial person created through a cloning-like process. The result is not a perfect copy and lacks the memories of the original. This app could be thought of us a not-very-good copy of the cells of a designer without any of their memories.
